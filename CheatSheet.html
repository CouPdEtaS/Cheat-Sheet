HTML Tags

Some HTML tags commonly used in the early lessons of Code Academy.

Common HTML Selectors
Tag Name Description
<div> Content Division Sub-divide and identify content. Also for CSS modification
<nav> Navigation Element Allows for navigation within a page or to an external page. Internal nav tags are acompanied by a # modifier.
<style> Style Element Style allows for in-line CSS style modification. Ex: <style="background-color: #FFFFF">Content<style>
<table> Table Contents Shows the contents of a table. Can be added to with various mods such as 'th', 'tr' etc.
<img> Image Link Links to an image either on your HD or on the web
<aside> Supplemental Content Will put content aside if it's relavent but not applicable to a particualr paragraph

CSS Attributes

CSS modifications that are commonly used in beginning CSS coding.

Common CSS Modifiers
Element Name Tree Branches
font: font subtree font-family, font-weight, font-size, font-style
Box Model Contents of a section to be displayed Box Model consists of Content, Padding, Border, and Margin. Each can be manipulated and adjusted.
color: Color of a Section or Element Color can be described by three values. Hex (#00000), RGBA (0,0,0), and declared: Color: ___ are all valid ways of describinbg color. You can apply using background-color to an element as well.
height:/width: Determines the pixel or percentage height/width of an object. can be written as height: 100% or width: 300px.
background-img: Image Link behind an object Looks like this:
body {
background-image: url("paper.gif");
}
